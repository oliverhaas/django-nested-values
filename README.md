# Django Prefetch Values

[![PyPI version](https://img.shields.io/pypi/v/django-prefetch-values.svg)](https://pypi.org/project/django-prefetch-values/)
[![CI](https://github.com/oliverhaas/django-prefetch-values/actions/workflows/ci.yml/badge.svg)](https://github.com/oliverhaas/django-prefetch-values/actions/workflows/ci.yml)

Enable `.prefetch_related().values()` in Django ORM - returns nested dicts with prefetched relations.

This package solves [Django ticket #26565](https://code.djangoproject.com/ticket/26565), which has been open since 2016.

See the [documentation](https://oliverhaas.github.io/django-prefetch-values/) for installation, usage, and API reference.
